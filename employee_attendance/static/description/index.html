Employee Attendance Report

About :-

 This Module is useful for Analyse the Employee attendance.
Employee Attendance module uses for analyse employee attendance Report like,
1)Daily
2)Weekly
3)Monthly
Using this module you can get data in PDF as well as Xls File format report. let's have demo for how to use this module.
Daily Basis :-

First of all, Click on Employee Attendance, select date,select daily from radio buttons and select All Employee check box so you can get all employee attendance detail on daily basis.


weekly Basis :-

when you select weekly option at that time you can get weekly report of attendance.


Monthly Basis :-

when you select Monthly option at that time you can get Monthly report of attendance.


Attendance report for specific employee.

select employee on employee filed and select radio button daily, weekly or monthly.


Download this report in two way
xls->press on xls button and get this pop on screen and press on this link.
