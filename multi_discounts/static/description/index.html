Multiple Discounts in Sale and Invoice.


In many cases we need to apply Discount on dicount but normal odoo does not support that so to have such functionality we have added this module. Using this module you can add discount on discount in Sale and Invoice lines. In Sale and Invoice Lines you can add multiple discount amounts seprated by '+'. If you apply '10+5+2' it will apply first 10% discount and then it will apply 5% on new amount and then it will apply 2%.
